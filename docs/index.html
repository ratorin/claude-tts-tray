Claude TTS Tray

Claude Code の応答を読み上げ、確認時に音を鳴らす常駐トレイ

Windows 版をダウンロードclaude-tts-tray.exe Linux 版 (x86_64)claude-tts-tray-linux-amd64

⚠️ Windows では未署名のため、初回起動時に SmartScreen の警告（「WindowsによってPCが保護されました」）が出ます。「詳細情報」→「実行」をクリックすると起動できます。

これは何？

Claude Code が応答を終えたときに最後の返答を読み上げ、確認（質問・許可など）が表示されたときに音を鳴らす常駐トレイアプリです。VSCode 拡張・Obsidian Claudian・ターミナルのいずれでも動作します（すべて claude CLI のフック経由のため）。

応答完了の読み上げ（Stop）
確認時の通知音（チャイム/発話）
読み上げと確認で別の声
今すぐ停止（トレイ左クリック）
VOICEVOX / AivisSpeech 対応
Windows / Linux 対応・単一バイナリ

Scoop でインストール（Windows）

パッケージマネージャ Scoop を使うと SmartScreen の警告なしで導入・更新できます。

scoop install https://raw.githubusercontent.com/ratorin/claude-tts-tray/main/scoop/claude-tts-tray.json

使い方

  • ダウンロードした exe を実行するとトレイに常駐します（右クリックでメニュー、左クリックで停止）。
  • 音声合成での読み上げには VOICEVOX または AivisSpeech（VOICEVOX互換API）の接続が必要です。未接続でも完了音・チャイムは鳴ります。
  • サーバー・話者・自動起動などの設定はトレイメニューから行えます。
